United States College Data 2013

A presentation of college data for 2013. To provide consitency, null valued schools are ignored and only public and private schools are considered (no for profits).

By Jon Alvarado
About the Data

The data set comes from the data.gov database. This data is compilation of different college stats compiled together to make it easier for prospective college students to decide their future schools. The original size of the data set is 7805 rows by 1730 columns.
No heavy duty processing went into filtering the data. Because I wasn't sure how the values were going to distribute over the graphs, I kept whole columns of only the statistics I thought I was going to use, such as SAT_SCORES, AVG_COST of attendence, RET_RATE of schools, etc.

Motivation

Raw data provided by source is unreadable at best. To make the information easier to digest for prospective college students, I've summarized the data provided through graphical representation.

Geographical Representation and Bar Graph Insights

Use the map to explore average rates, costs, and scores for all 50 states and DC. The bar
graph presents the top 10 for a certain average. Zooming into a state provides insight into
the schools within the state which is further reflected in the bar graph which updates to
present the top 10 schools within a state for a certain statistic.

The scatter plot is to be used as a summary of all the information in one graph. It provides
a view of all the average touch points while staying relevant to the selected data view by reflecting the same color coded information.
